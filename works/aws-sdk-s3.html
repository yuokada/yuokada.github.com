Proxy¶

  • rhelmer/caching-s3-proxy: provides an unauthenticated plain HTTP frontend for public and private S3 buckets, and caches on the filesystem using LRU

  • azavea/docker-s3-proxy-cache: A proof of concept HTTP caching proxy for requests against Amazon S3. これ有力そう。

  • Nginx S3/Unicorn Proxy with backend keep alive

  • Fungoing Labs: Amazon S3のファイルを国内VPS（nginx）にキャッシュして配布

  • coopernurse/nginx-s3-proxy: nginx compiled with aws-auth support, suitable for S3 reverse proxy usage

  • anomalizer/ngx_aws_auth: nginx module to proxy to authenticated AWS services
